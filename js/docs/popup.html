Popup Class Manual

The Popup class allows you to create customizable popups on your web page. You can easily open and close popups with various options to control their appearance and behavior.

Usage

  1. Create an instance of the Popup class.
  2. Customize the popup options.
  3. Open the popup using the open() method.
  4. Close the popup using the close() method.

Example

const myPopup = new Popup();

myPopup.options = {
  id: 'my-popup',
  title: 'Hello World',
  content: 'This is the content of the popup.',
  closeButton: true,
  buttons: [
    { text: 'OK', className: 'btn-ok', onClick: () => { /* Handle OK button click */ } },
    { text: 'Cancel', className: 'btn-cancel', onClick: () => { /* Handle Cancel button click */ } }
  ]
};

myPopup.open();
myPopup.close();

Options

Option Type Default Description
id string 'popup' The ID of the popup element.
title string '' The title of the popup.
content string '' The content of the popup.
closeButton boolean true Specifies whether to show a close button in the popup header.
buttons array [] An array of objects representing buttons in the popup footer. Each button object should have text, className, and onClick properties.
zIndexBase number 9900 The base z-index value for the popup and backdrop elements.
timer number or null null The duration in milliseconds after which the popup should automatically close.
beforeOpen function empty function A callback function called before the popup is opened.
afterOpen function empty function A callback function called after the popup is opened.
beforeClose function empty function A callback function called before the popup is closed.
afterClose function empty function A callback function called after the popup is closed.

CSS Classes

  • popup: The main container for the popup.
  • popup__header: The header section of the popup.
  • popup__title: The title element within the popup header.
  • popup__content: The content section of the popup.
  • popup__footer: The footer section of the popup.
  • popup__button: The button elements within the popup footer.
  • popup__close: The close button in the popup header.
  • popup__backdrop: The backdrop element behind the popup (optional).
  • popup__backdrop--transparent: Additional class for transparent backdrop (optional).
  • popup--modal: Class added to the body when a modal popup is displayed (optional).
